# Notifications app

![image](https://github.com/dayanapanova/notifications-app/assets/67751911/bc32551e-a898-44fa-ad2a-5957173d2a10)


## Start database

```shell
docker-compose up database
```

## Install packages

```shell
pnpm install
```

## Start dev server

```shell
pnpm dev
```

## Requirements

- Node >= 18.0.0
